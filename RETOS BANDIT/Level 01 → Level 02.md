# Objetivo del Nivel
The password for the next level is stored in a file called **-** located in the home directory
# Datos de Acceso Al Nivel
bandit1 NH2SXQwcBdpmTEzi3bvBHMM9H66vVXjL
# Solución

```bash
bandit1@bandit:~$ ls
-
bandit1@bandit:~$ cat -
^C
bandit1@bandit:~$ cat ./-
rRGizSaX8Mk1RTb1CNQoXTcYZWU6lgzi
bandit1@bandit:~$ 
```
